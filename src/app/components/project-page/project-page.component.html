Project manager

A full stack web app using Java Spring Boot backend REST API, Angular frontend with Bootstrap stylings and MySQL database.
Hosted on a DigitalOcean VPS.
Registered users can add, remove and edit their personal projects.
Projects can have names, descriptions, dates, statuses, tasks and images.
